LockWise AI

LockWise AI is a WPF-based cybersecurity awareness chatbot developed in C#. The application was created to educate users about online safety through interactive conversations and helpful cybersecurity tips.

The chatbot provides advice on topics such as:

Password safety
Phishing awareness
Safe browsing
General cybersecurity practices
## Features
*Modern WPF Graphical User Interface
Professional and user-friendly design
Interactive chat layout
Styled buttons and message bubbles

*Keyword Recognition

The chatbot recognizes keywords related to cybersecurity topics such as:

Passwords
Phishing
Browsing

*Random Responses

The chatbot provides different responses for certain topics to make conversations feel more natural and less repetitive.

*Memory Feature

The chatbot remembers the user's name during the conversation and greets them personally.

*Sentiment Detection

The chatbot can detect simple emotional keywords such as:

sad
stressed

and responds supportively.

*Typing Effect

A typing effect was added to simulate a more realistic chatbot interaction.

*Auto Scrolling Chat

The chat window automatically scrolls to the latest message.

*Enter Key Support

Users can press the Enter key to send messages instead of clicking the Send button.
